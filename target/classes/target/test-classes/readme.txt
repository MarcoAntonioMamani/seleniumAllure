
Peerequisito tener configurado Maven y el jdk

1.- Descargar esta carpeta y pegarla en su pc, por ejemplo D://QADevsuMarco

2.- Abrir power sheel e instalar allure

>> scoop install allure

3.- Luego dirigirse a la carpeta que en este ejemplo es : "D://QADevsuMarco" y ejecutar el comando 

D://QADevsuMarco>> mvn clean test

4.- Finalmente ejecutar 

D://QADevsuMarco>> allure server
